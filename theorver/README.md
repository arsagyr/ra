## Описание пакета
Пакет theorver определяет функции из области теории вероятностей.

## Реализованнные функции
Число сочетаний
Число разупорядочиваний
Коэффициент Фи
Функция Бернулли
Локальная теорема Муавра-Лапласа
Интегральная теорема Муавра-Лапласа

## Использование

Ниже представлены примеры применения функций. **Внимание: не забывать о возможности перезаполнения при вычислении больших значений** 

### Число сочетаний

```go
    var c int64
    c = Comb(a, b)// Возвращает число сочетаний переменных a и b типов данных int, int32 и int64
```

### Число разупорядочиваний
```go
    var d int64
    d = Discomb(n)//Возвращает число сочетаний переменных n типа данных int
```

### Функция Бернулли
```go
    var b float64
    b = Bernulli(n, k, p)//Возвращает значение функции Бернулли для переменных a, b типа данных int и p типа данных float32
```

### Коэффициент Фи
```go
    var phi float64
    phi = PhiCoef(x)//Возвращает значение коэффициента Фи для x типов данных int, int32 и int64
```

### Интегральная теорема Муавра-Лапласа
```go
    var f float64
    f = Integral_De_Moivre_Laplace_theorem(n, p, a, b)//Возвращает значение интегральной теоремы Муавра-Лапласа для n типа данных int, p, a и b типа данных float32
```

### Локальная теорема Муавра-Лапласа
```go
    var f float64
    f = Local_Fucntion_Moivre_laplace(n, p, ik)//Возвращает значение локальной теоремы Муавра-Лапласа для n int, p float32, ik int
```
