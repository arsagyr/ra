# RA GO
RA GO - Reckoning Algorithms of GO - английское наименование, русское название - "Расчётные Алгоритмы GO -РА GO" - система для введения математических рассчётов на ЯП Golang для системы SARA GO - САРА GO.

Здесь представлены пакеты математических моделей. Пакет работы с дробями - `fraction`, содержит структуру `Fraction` с определенными для себя методами для создания дробей, их печати и выполнения арифметических операций. Пакет работы с дискретными случаными величинами - `drv`, содержит структуру `DRV` с определенными для себя методами для создания дискретных случайных величин, их печати и выполнения операций над ними. Пакет вызова функций из области теории вероятностей - `theorver`.

## Возможности

- Создание дробей с использованием числителя и знаменателя.
- Печать дробей в стандартном формате.
- Умножение дробей на целые числа и другие дроби.
- Создание случайных величин с использованием дроби.
- Печать случайных величин и их вероятностей.
- Выполнение арифметических операций над случайными величинами.
- Реализация функций из области теории вероятностей.

## Установка

Чтобы использовать эту библиотеку, убедитесь, что у вас установлен Go. Вы можете клонировать репозиторий и запустить код напрямую.

```bash
git clone https://github.com/arsagyr/ra
```

Или импортировать в проект

```go
import "github.com/arsagyr/ra"
```
Не забудьте обновить зависимости

```bash
go mod tidy
```
